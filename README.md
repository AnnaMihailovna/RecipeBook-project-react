# praktikum_new_diplom

На даный момент проект настроен для локального запуска
без контейнеров.

Проект разрабатывался на SQLite.
Следующие шаги: подключение PostgreSQL,
                создание образов,
                запуск контейнеров,
                деплой на удаленный сервер через GitHub Actions.
